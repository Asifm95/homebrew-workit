# homebrew-workit

Homebrew tap for [workit](https://github.com/Asifm95/workit) — a CLI workflow manager for multi-project git worktrees.

## Install

```sh
brew tap Asifm95/workit
brew install workit
```

Or in one line:

```sh
brew install Asifm95/workit/workit
```

## Upgrade

```sh
brew update
brew upgrade workit
```
